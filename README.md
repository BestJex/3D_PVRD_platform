### wepy框架

微信小程序端代码待用wepy框架，测试运行需要先npm安装wepy-cli，运行

``` js
wepy build --watch
```

生成dist文件夹后，可通过微信开发者工具预览

### wepy-zanui

微信小程序端UI组件采用wepy-zanui组件库
